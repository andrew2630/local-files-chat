Local Files Chat macOS portable (unsigned)

Unzip the archive, then move Local Files Chat.app anywhere (for example, /Applications).

If macOS blocks the app:
- Right-click the app and choose Open, then confirm.
- Or remove quarantine:
  xattr -dr com.apple.quarantine "/path/to/Local Files Chat.app"

Optional:
- Run RemoveQuarantine.command in this folder to clear quarantine for the app next to it.
